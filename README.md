SwarMapper: Western to Indian Classical Music Translator
SwarMapper is a dynamic web-based utility designed to bridge the gap between Western Chromatic notation and Indian Classical (Hindustani) Swaras. Since Indian music relies on a relative pitch system (where 'Sa' is movable), this tool allows musicians to define their root note and instantly see the corresponding 12-note chromatic mapping.

✨ Features
Dynamic Root Selection: Set any of the 12 Western notes as your 'Sa' (Shadaj).

Full 12-Note Chromatic Mapping: Includes all Shuddh (Natural), Komal (Flat), and Teevra (Sharp) notes.

Responsive Design: Clean, mobile-friendly interface for use during practice sessions or studio work.

Educational Tooltips: Displays the full traditional names of the Swaras (e.g., Komal Rishabh, Teevra Madhyam).
